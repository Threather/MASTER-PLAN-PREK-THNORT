# Prek Thnort — Land Map

Two files. One for drawing plots, one for the live map with Firebase sync.

---

## Files

| File | Purpose |
|------|---------|
| `editor.html` | You use this to draw plot shapes. Export `plots.json` when done. |
| `viewer.html` | The live map. Load `plots.json`, click plots to change color. Syncs to Firebase. |
| `plots.json` | Your plot shapes. Generated by editor.html. |

---

## Step 1 — Set up Firebase (free, 5 minutes)

1. Go to **https://console.firebase.google.com**
2. Click **Add project** → name it (e.g. `prek-thnort`) → Create
3. On the left menu click **Build → Realtime Database**
4. Click **Create Database** → choose any location → Start in **test mode** → Enable
5. On the left menu click **Project Settings** (gear icon ⚙️)
6. Scroll down to **Your apps** → click `</>` (Web app)
7. Give it a name → click **Register app**
8. You will see a config block like this:

```js
const firebaseConfig = {
  apiKey: "AIza...",
  authDomain: "prek-thnort.firebaseapp.com",
  databaseURL: "https://prek-thnort-default-rtdb.firebaseio.com",
  projectId: "prek-thnort",
  storageBucket: "prek-thnort.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123"
};
```

9. Open `viewer.html` in a text editor (Notepad, VS Code, etc.)
10. Find the section that says `▼▼▼ PASTE YOUR FIREBASE CONFIG HERE ▼▼▼`
11. Replace the placeholder values with your real config values
12. Save the file

---

## Step 2 — Draw your plots

1. Open `editor.html` in your browser (just double-click it)
2. Click corners around each plot to trace its shape
3. Double-click (or click the first green dot) to finish each plot
4. Rename plots in the side panel if needed
5. Click **💾 Export plots.json**
6. To add more plots later: open editor → Load plots.json → draw more → export again

---

## Step 3 — Host on GitHub Pages (free)

1. Go to **https://github.com** → New repository → name it `prek-thnort-map`
2. Upload these files to the repo:
   - `editor.html`
   - `viewer.html`
   - `plots.json`
   - `README.md`
3. Go to **Settings → Pages**
4. Under **Source** select `main` branch → Save
5. Your map will be live at: `https://YOUR-USERNAME.github.io/prek-thnort-map/viewer.html`

Share that link with your team. Anyone who opens it will see the same map, and changes sync instantly via Firebase.

---

## How it works day to day

1. Open `viewer.html` (local or GitHub Pages link)
2. Click **Load plots.json** once
3. Click any plot → choose ⬜ Unsold / 🟡 Reserved / 🔴 Sold
4. Firebase updates instantly — everyone sees the change in real-time

## Firebase security (when ready)

Right now the database is in **test mode** (open to everyone).  
When you're ready to lock it down, go to Firebase → Realtime Database → Rules and set:

```json
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
```

For proper auth, contact a developer to add Firebase Authentication.
